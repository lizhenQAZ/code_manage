这是and操作符:

{% for a in alist %} {% if a and True %}
第{{ forloop.counter }}次循环, 值为真
{% else %}
第{{ forloop.counter }}次循环, 值不为真
{% endif %} {% endfor %}

这是or操作符:

{% for a in alist %} {% if a or False %}
第{{ forloop.counter }}次循环, 值为真
{% else %}
第{{ forloop.counter }}次循环, 值不为真
{% endif %} {% endfor %}

这是not操作符:

{% for a in alist %} {% if not a %}
第{{ forloop.counter }}次循环, 值不为真
{% else %}
第{{ forloop.counter }}次循环, 值为真
{% endif %} {% endfor %}
